# SeQwens Step-by-Step Guide

## **Phase 1: Getting Started**
- **Login**: Go to SeQwens website → Enter email/password → Set up security if first time.
- **Dashboard**: After login, you'll see main dashboard. Use top bar for navigation.
- **Roles**: Your role (Super Admin, Firm Admin, etc.) controls what you see.
- **Firm Setup**: Firm Admin → Top menu "Settings" → "Firm Profile" → Fill company details.
- **Account Settings**: Click your name (top right) → "Settings" → Update info, password, notifications.
## **Phase 2: Team Setup**
- **Invite Team**: Firm Admin → Top menu "Team" → "Invite Member" → Enter email/role → Send.
- **Assign Roles**: During invite or in "Team" section → Edit member → Change role.
- **Manage Profiles**: "Team" section → View/edit member details or deactivate.
- **Assign Clients**: Client profile → "Assign Team Member" dropdown → Select people.
- **Auto Assignment**: "Settings" → "Client Assignment" → Set rules (like workload balancing).

## **Phase 3: Client Management**
- **Add Client**: Top menu "Clients" → "Add Client" → Fill form → Save.
- **Send Invite**: Client profile → "Send Portal Invitation" → Email sent automatically.
- **Portal Access**: Clients get email link → Login to upload documents.
- **Bulk Import**: "Clients" → "Import" → Download CSV template → Fill data → Upload.
- **Fix Errors**: Check error report after import → Correct CSV → Re-upload.

## **Phase 4: Client Organization**
- **Add Tags**: Client profile or bulk edit → Add labels like "Priority".
- **Edit/Remove Tags**: Client details → Click tag → Change/delete.
- **Search/Filter**: Top search bar → Type name/email → Apply filters (tags, status).
- **Activity History**: Client profile → "Activity" tab → View timeline.
- **Archive Clients**: "Clients" → "Archived" → Restore or delete.

## **Phase 5: Document Management**
- **Request Documents**: Client profile or "Tasks" → Create request → Set types/dates.
- **Client Uploads**: Clients see requests in portal → Upload via drag/drop.
- **Internal Upload**: "Documents" tab in client profile → Upload file → Add notes.
- **Organize**: Use folders/tags in "Documents" → Search to find files.
- **Track Missing**: "Reports" → "Missing Documents" → Send reminders.

## **Phase 6: Tasks & Work**
- **Create Task**: "Tasks" → "New Task" → Fill details → Assign.
- **Assign/Track**: Task edit → Change assignee → Update status/notes.
- **Due Dates/Priorities**: Task creation → Pick calendar/priority level.
- **Complete**: Mark task "Done" → Add notes → Archive.

## **Phase 7: Workflow Automation**
- **Intro**: Workflows automate repeated processes.
- **Create Template**: "Workflows" → "New" → Set triggers/steps.
- **Assign**: Client profile → Apply workflow, or set auto-triggers.
- **Auto Tasks**: Triggers create tasks automatically in sequence.
- **Tag Triggers**: Configure workflows to start on specific client tags.

## **Phase 8: Communication**
- **Send Messages**: Client profile → "Messages" → Compose/send.
- **View Conversations**: "Messages" tab → See history.
- **Notifications**: "Settings" → Choose alerts (email/SMS).
- **Email Integration**: Messages use firm branding → Clients reply in threads.

## **Phase 9: E-Signature**
- **Send for Signature**: Upload document → Use "E-Signature" → Mark fields → Send.
- **Client Signing**: Clients get email → Portal → Sign electronically.
- **Track Status**: Document details → See "Sent/Viewed/Signed".

## **Phase 10: Monitoring**
- **Activity Logs**: Client profile → "Activity" → View all logs.
- **Team Workload**: Dashboard widgets → Check assignments/completion.
- **Best Practices**: Regular reviews, consistent tagging, workflow use, team training, updates.

**Quick Tips**:
- Start from dashboard after login.
- Use top navigation bar for most actions.
- If stuck, check help section or contact support.
- Always save changes and use search for quick finds.
